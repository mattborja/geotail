# data/

Drop your IP2Location LITE BIN files here — geotail finds them automatically:

- `IP2LOCATION-LITE-DB11.BIN` (geolocation, required)
- `IP2PROXY-LITE-PX11.BIN` (proxy/VPN/Tor, optional)

Download both for free from <https://lite.ip2location.com> (account required, no credit card).
BIN files are gitignored; only this note is committed.
